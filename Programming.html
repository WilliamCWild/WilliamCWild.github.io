Automated Retail Partner DB Update

Retail partnership contract updates would pull from internal database to PythonAnywhere server at 98% cost reduction vs previous server hosting method. Retail partner data would then transfer to email database via API connection for accurate and timely communication of vital business information.

Pricing Accuracy Tool

Scripted price & promo checking to ensure retail partner agreements were being upheld and honored consistently. The program would pull thousands of SKU prices from internal database via API connection, connect to owned eCommerce platform to cross-reference public-facing prices, compare to retailer websites with same SKUs for sale, and finally output a CSV of any mispriced or missing SKUs.
